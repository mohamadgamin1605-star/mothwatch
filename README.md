# Mothwatch

FreeFlix Hub is a streaming application for watching free, licensed movies and anime.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

This is a TanStack Start TypeScript project with:
- React 19
- TanStack Router for routing
- TanStack React Query for data fetching
- Tailwind CSS for styling
- Radix UI components
